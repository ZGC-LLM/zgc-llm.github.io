## 9. 需在飞书 UI 手动收尾（CLI 不支持的部分）

以下为飞书 OpenAPI/CLI 无法配置、只能在表单编辑界面完成的项：

1. **隐藏审核字段**：两个表单当前把审核类字段（审核状态/审核人/审核意见/审核时间等）也显示为题目——在表单编辑器里把它们从表单移除（仅隐藏，不删列）。注意不要用 CLI 删除，CLI 删除会连同表格列一起删掉。
2. **条件显示（分流）**：
   - 入会表：「申请级别/申请主体类型 = 个人」时显示个人字段（研究/从业方向、推荐人），隐藏单位字段（统一社会信用代码、机构类型、机构简介及主营方向）；反之亦然。
   - 工作组表：「是否已入会 = 已是会员/已同时提交」时显示"已入会机构名称/申请编号"；「申请主体类型 = 单位」时显示统一社会信用代码。
3. **核对联系电话/联系邮箱**：phone/email 样式字段未被 CLI 列为表单题目，请确认其在表单上正常出现；若缺失，将字段改为纯文本或在 UI 手动加入。
4. **题目排序**：按问卷终稿顺序在 UI 拖拽调整。
5. **发布并获取分享链接**：表单发布后取分享链接，供官网两个入口跳转。
