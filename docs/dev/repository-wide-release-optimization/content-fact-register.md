# 公开内容事实与授权登记

本登记把“事实是否可核实”和“是否获准在官网公开/复用”分开。公开网页、仓库注释、`published:true`、`named:true`、现有测试、可访问的 Feishu URL 都不能自动证明授权；GET 200 也不能证明匿名用户能完成表单。中文是事实权威源，英文只能传达等价信息，不能新增承诺或擅自创造官方译名。

## 1. 裁决规则

| 状态            | 含义                                                      | 默认发布动作                                            |
| --------------- | --------------------------------------------------------- | ------------------------------------------------------- |
| 已有公开来源    | 至少有可访问的公开来源直接支持该名称/数字/职务            | 只在来源覆盖的范围内保留；不外推 logo、英文名或更广关系 |
| 项目权威决定    | AGENTS/配置明确项目希望采用的口径，但外部运营状态尚未形成 | 可作为配置目标；对“已上线/可访问”类陈述仍需外部复验     |
| 仓库声明        | 只有内容常量、注释、测试或历史规格支持                    | 不是授权；涉及背书/伙伴/结果时阻塞或中性化              |
| 部分佐证        | 第三方/合作方来源支持部分范围，但不是联盟当前运营确认     | 只保留明确被佐证的范围并标日期；承诺、授权仍待确认      |
| 冲突            | 仓库/外部入口之间存在互斥表述                             | 发布阻塞，先由事实 owner 作唯一裁决                     |
| 未核实/外部阻塞 | 缺来源、权限、DNS、匿名流程或当前运营证据                 | 保留会阻塞发布；否则从公开页面移除或换中性表达          |
| 编辑分类        | 数字只是页面分类/清单结构，不是绩效或结果                 | 可作为栏目结构，不得改写为“已取得 N 项成果”             |

授权范围统一分为：`名单文字`、`职务/合作角色`、`官方英文名`、`logo/品牌资产`、`成果/承诺`。一个范围有证据不代表其他范围也有证据。
